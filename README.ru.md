# Кастомный ремап стрелок / Copilot / Shift (AutoHotkey v2)

[English version / Английская версия](README.md)

## Зачем

На моем новом ноутбуке постоянно вместо стрелки «Влево» нажимаю соседнюю Copilot. Она открывает панель на пол экрана и мешает. Я перебиндил клавиши, чтобы вернуть привычную навигацию и убрать ложные вызовы Copilot.

## Что изменено

| Физическая клавиша | Теперь выполняет | Примечание |
|--------------------|------------------|------------|
| Copilot (Win+Shift+F23) | Стрелка Влево | Становится единственной «Влево». |
| Правый Shift | Стрелка Вверх | Вверх теперь дублируется: обычная ↑ + Right Shift. |
| Стрелка Вниз | Стрелка Вправо | Вправо дублируется: обычная → + бывшая ↓. |
| Стрелка Влево | Стрелка Вниз | «Вниз» переехала на место «Влево». |

Итог:
- Вверх: обычная ↑ и Right Shift.
- Вправо: обычная → и бывшая ↓.
- Вниз: бывшая ←.
- Влево: Copilot.
(Оригинальные ↑ и → не тронуты.)

## Требования

- Windows 10 / 11
- [AutoHotkey v2](https://www.autohotkey.com/)

## Установка AutoHotkey

1. Перейдите на сайт [AutoHotkey](https://www.autohotkey.com/).
2. Нажмите Download → Download v2.
3. Установите.

## Скрипты

Два варианта:
1. С F12 (быстрый выход).
2. Без F12 (минимализм).

Сохраните выбранный как `remap_arrows.ahk`.

### Вариант 1 (с F12)

```ahk
#Requires AutoHotkey v2.0
#SingleInstance Force
*<#<+F23::{
    Send "{Blind}{LShift up}{LWin up}"
    Send "{Left}"
    Sleep 400
    while GetKeyState("F23","P"){
        Send "{Left}"
        Sleep 30
    }
}
*RShift::{
    Send "{Up}"
    Sleep 400
    while GetKeyState("RShift","P"){
        Send "{Up}"
        Sleep 30
    }
}
*Down::{
    Send "{Right}"
    Sleep 400
    while GetKeyState("Down","P"){
        Send "{Right}"
        Sleep 30
    }
}
*Left::{
    Send "{Down}"
    Sleep 400
    while GetKeyState("Left","P"){
        Send "{Down}"
        Sleep 30
    }
}
F12::ExitApp
```

### Как работает

- Copilot шлет Win+Shift+F23. Скрипт сначала «отпускает» Win и Shift, чтобы не было выделения или привязки окна.
- Первая отправка — одно нажатие, затем после паузы (`Sleep 400`) начинается быстрый повтор (`Sleep 30`).
- Остальные клавиши — аналогичный цикл автоповтора.

### Настройка скорости

- `Sleep 400` — задержка перед серией (имитация стандартной клавиатурной задержки).
- `Sleep 30` — интервал между повторными нажатиями (меньше = быстрее).

## Запуск

Двойной клик по `.ahk` файлу. Зелёная иконка H в трее — скрипт активен.

## Автозапуск

1. `Win + R`
2. Ввести: `shell:startup` → Enter
3. Положить ярлык `.ahk` файла в открывшуюся папку.
4. После входа в систему скрипт стартует сам.

## Советы

- Если игнорируется в каком‑то приложении — запустите от имени администратора.
- Перезагрузка после правки: правый клик по иконке → Reload Script.
- В варианте с F12 можно быстро отключить.

## Отказ от ответственности

Скрипт меняет поведение клавиш только пока запущен. Закрыли — всё штатно. Используйте на свой страх и риск (он безопасен, просто нестандартен).
